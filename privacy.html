Политика обработки персональных данных

Эта политика описывает обработку данных, которые посетитель передает через форму заявки на сайте Pony RIDE.

Оператор

Оператор персональных данных: Индивидуальный предприниматель Макарушина Марина Юрьевна.

ИНН: 27604311389. ОГРНИП: 324028000011292.

Контакт для обращений: ponyride_official@mail.ru.

Какие данные собираются

  • имя и телефон;
  • выбранная модель, срок аренды и цель обращения;
  • признак обращения из другого города Республики Башкортостан и указанный город;
  • комментарий, если пользователь его добавил.

Цель обработки

Данные используются для обработки заявки, связи с клиентом, уточнения модели, срока аренды, времени выдачи и условий аренды с правом выкупа.

Как обрабатываются данные

Данные передаются только после отправки формы пользователем. Сайт не сохраняет персональные данные в localStorage или sessionStorage.

Если в конфигурации сайта подключен endpoint Google Apps Script, форма передает данные этому стороннему endpoint для приема заявки. Оператор должен настроить доступ и хранение заявок в соответствии со своими процессами.

Сторонние сервисы

При подключении карты на сайте может загружаться iframe стороннего картографического сервиса. Если карта не настроена, сайт показывает карточку с адресом и внешнюю ссылку на карту.

При настройке Яндекс Метрики сайт может отправлять обезличенные события интерфейса счетчику аналитики. Без ID счетчика код Метрики не подключается.

Права пользователя

Пользователь может обратиться по email оператора, чтобы уточнить порядок обработки своих данных или отозвать согласие в пределах применимого законодательства.
